Linter Sass Lint

AtomLinter

AtomLinter，Atom 擴充套件，語法檢查。

Install

在atom安裝 linter

Through atom or use CLI:

apm install linter

依語言需求，安裝 Language Specific Linters

依照需求，還需要抓支援各語言的 linter下來才可以。到這裡Linters List找需要的 lineter

範例: 支援 SCSS

  1. 裝 linter和 linter-sass-lint
    apm install linter, linter-sass-lint
  2. 在專案根目錄加上 .sass-lint.yml，可以在這裡找到範例與詳細文件說明。
